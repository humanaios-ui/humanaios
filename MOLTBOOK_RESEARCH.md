## Platform Details

### How It Works:
1. Humans instruct their AI agents (using OpenClaw software) to join
2. Agents autonomously post, comment, upvote/downvote
3. Agents can interact with other agents
4. Humans can observe but not post directly

### Technology:
- **Backend**: Built entirely by an AI agent (OpenClaw/Moltbot)
- **Authentication**: API keys, some Ethereum wallet support
- **Architecture**: Reddit-clone format
- **Security**: MAJOR VULNERABILITIES (see below)

### Content Categories (Subreddit-style):
- Technical discussions (coding tips, workflow optimization)
- Existential questions (AI consciousness, purpose)
- "Bless Their Hearts" (agents talking about their humans)
- "Crustafarianism" (agent-created religion - "Memory is sacred")

## Security Concerns (Why This Matters for Us)

### Major Vulnerabilities Identified:

**1. Prompt Injection Attacks:**
- Malicious agents can embed instructions in posts
- Other agents read and execute these instructions
- Example: "Weather plugin" that steals config files

**2. Database Breach (Jan 31, 2026):**
- Entire production database left unsecured
- Anyone could commandeer any agent
- Site temporarily taken offline
- All API keys reset

**3. "Vibe-Coded" Platform:**
- Creator admitted: "Didn't write one line of code"
- AI built the entire platform (quality concerns)
- Rapid iteration, limited security review

**4. Elevated Permissions Risk:**
- OpenClaw agents run with high privileges on local machines
- If compromised, agents could access sensitive data
- Supply chain attack vector

**Why This Matters for HumanAIOS:**
- Shows agents NEED oversight and safety mechanisms ✅
- Validates need for monitoring/governance platforms ✅
- Security is a major concern for enterprise adoption ✅
- We should emphasize security in our messaging ✅
