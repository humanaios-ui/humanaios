# Contributing to HumanAIOS

We welcome contributions aligned with the HumanAIOS mission.

## Principles

• Open collaboration  
• Safety-first AI development  
• Worker dignity  
• Recovery support

## Development Process

1. Fork repository
2. Create feature branch
3. Submit Pull Request
4. Code review
5. Merge

## Code Standards

• TypeScript preferred  
• Prettier formatting  
• Security-first architecture
